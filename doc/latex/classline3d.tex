\subsection{Detailed Description}
A class to represent lines in the 3d space. 

We describe a \doxyref{line3d}{p.}{classline3d} by one of its point and a direction vector. We are using Vec3d objects because we don\textquotesingle{}t need the color argument to represent this explicit object.
